How to Use This Site

Stats: Select a country and a date range to view total COVID-19 cases and deaths.

Compare: Choose 2 - 5 countries and a specific date to compare their case and death counts.

Use the calendar input to select valid dates.

If you see 0 cases and deaths, it may mean no data is available for that selection.
